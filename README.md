# Oltao - Next.js E-Commerce Project

Welcome to the Oltao e-commerce project! This modern, responsive web application is built with [Next.js](https://nextjs.org), designed to showcase premium fans and home appliances with a focus on visual aesthetics and smooth user interactions.

## üöÄ Key Features

- **Modern Tech Stack**: Built with Next.js 16 (App Router), React 19, and TypeScript.
- **Responsive Design**: Fully responsive layout optimized for all devices, from large desktops to mobile screens.
- **Animations**: Smooth, staggered entrance animations and hover effects powered by `framer-motion`.
- **Styling**: Utilizes Tailwind CSS v4 for rapid, utility-first styling and a custom design system.
- **Interactive Components**: Includes a custom cursor, image gallery with thumbnails, animated feature grids, and FAQ accordions.

## üì¶ Getting Started

Follow these steps to set up and run the project locally.

### Prerequisites

- Node.js (v18 or higher recommended)
- npm, yarn, pnpm, or bun

### Installation

1.  **Clone the repository:**

    ```bash
    git clone <repository-url>
    cd oltao-nextjs
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```

### Running the Development Server

Start the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## üìÇ Project Structure

- `app/`: Main application code (App Router).
  - `components/`: Reusable UI components (Navbar, Footer, Main, ExploreProducts, WhyChooseUs, etc.).
  - `globals.css`: Global styles and Tailwind directives.
  - `layout.tsx`: Root layout component.
  - `page.tsx`: Home page.
- `public/`: Static assets (images, icons).
